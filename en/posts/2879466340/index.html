What is Helm

  • Helm is a package manager for Kubernetes

What is Helm chart

  • Helm chart is a bundle of YAML files
  • We can create our own helm chart and push them to the helm repository
  • Also we can download and use existing chart in helm repository

What is Tiller

Tiller is a server-side helm in k8s clusters that helps install the package in the cluster.

But Tiller has too much power inside the k8s cluster, so it may have many security issues. In helm3, Tiller is deprecated.
